Bookmarklet Setup

One-time config. Takes about 15 minutes.

01

Host the capture page

pavoi-capture.html needs to be accessible from a URL so the bookmarklet can open it on any device. Two easy options:

Option A — GitHub Pages (free, 5 min):
Create a public GitHub repo → upload pavoi-capture.html → enable Pages in repo Settings. Your URL will be https://YOUR-ORG.github.io/pavoi-inspire/pavoi-capture.html.

Option B — Shopify:
Upload to your Shopify theme's Assets folder. URL: https://pavoi.com/cdn/shop/t/X/assets/pavoi-capture.html.

Once hosted, paste the URL in Step 3 below.

02

Create your Google Form

Go to forms.google.com → New blank form. Add these fields in order (all Short Answer unless noted):

URL ★ Platform (Dropdown) Content Type (Dropdown) Brand Price Review Score Review Count Likes Comments Saves Saved By ★ Save Date Inspiration Note (Paragraph) ★ Tags Urgent (Dropdown: Yes / No)

After creating the form, click ⋮ → Get pre-filled link. Fill every field with a dummy value. Click "Get Link". Copy the URL. Paste it somewhere temporary — you'll extract the entry.XXXXXXXXX IDs from the URL parameters. Each entry.ID matches a field in order.

The form action URL is the same base URL with viewform?... replaced by formResponse.

04

iOS Shortcut (iPhone Share Sheet)

This lets you capture from Instagram, TikTok, or Safari with one tap via the native Share Sheet. Open the Shortcuts app → tap + to create a new shortcut.

  1. Add action: Receive — set input to URLs from Share Sheet
  2. Add action: URL — set value to:
    https://YOUR-CAPTURE-PAGE-URL?url=
  3. Add action: Combine Text — combine the URL from Step 2 with the Shortcut Input variable (URL-encoded)
  4. Add action: Open URLs — input = combined text from Step 3
  5. Name the shortcut PAVOI Save and set an icon
Simpler alternative Use the iOS Share Sheet → Safari → share to a Slack channel or email alias, then paste the URL into pavoi-capture.html manually on desktop. Works fine for a small team.
05

Publish your Google Sheet for the synthesis app

In the Google Sheet linked to your form: File → Share → Publish to web. Choose the response tab, format: CSV. Click Publish. Copy the URL.

Paste that URL into pavoi-synthesis.html in the SHEET_CSV_URL config field at the top of the file.

Note: Published CSV URLs are readable by anyone with the link. Keep this in mind for sensitive brand data. If privacy is a concern, use the "paste CSV" option in the synthesis app instead.
